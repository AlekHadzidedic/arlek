# Arlek

Source for [arlek.ca](https://arlek.ca) — a one-person studio in Ottawa building websites and automations for Canadian small businesses.

Next.js (App Router, TypeScript) + Tailwind v4, deployed on Vercel.

## Develop

```bash
pnpm install
pnpm dev      # http://localhost:3000
```

## Build & test

```bash
pnpm build    # production build (type-checked)
pnpm test     # Vitest + React Testing Library
```

## Deploy

Push to the connected Vercel project — zero-config Next.js build.
